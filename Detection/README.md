Использование ранее обученных LBP каскадов для поиска целевых объектов.

Программа загружает все найденные в указанных директориях каскады и изображения, загуржает каскады, откарывает изображения, пропроционально сжимает их, выполняет на них поиск объектов и помечает результаты.
